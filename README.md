## 진행 순서

라이브에서 아래 순서로 따라갑니다. **위에서 아래로**.

## 라이브 전 준비

[setup/README.md](./setup/README.md) 를 따라 4가지를 미리 설치해주세요.

- VSCode (코드 작업창)
- Python 3.11+ (실행 환경)
- Git (이력 도구)
- GitHub 계정 (웹에서 가입만)

설치가 끝까지 안 되어도 라이브엔 그대로 오시면 됩니다. 같이 봐드릴 수 있어요.
