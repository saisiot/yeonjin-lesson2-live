# 도전 과제 — 본인 시간에, 손으로

> 강의 끝나고 본인 시간에 풀어보는 빈칸 채우기입니다.
> **외울 거 X.** "이 한 줄을 직접 쳐보면 되네" 감만 잡으면 끝.

---

## 푸는 방법

1. 한 문제씩 위에서부터.
2. 막히면 **파일 하단의 정답을 살짝** 본다 (몇 개 파일은 정답이 같이 들어 있음).
3. 정답 보고 끄덕였으면, **정답 가리고 본인 손으로 다시** 친다.
4. 결과가 같이 나오면 OK. 다음 문제로.
5. 30분 막히면 그냥 다음 회차에 같이 보기. 혼자 끙끙 X.

> "코드의 코도 모르는 분들도 따라올 수 있게" — SRT 톤 그대로.

## 정답 위치

- 대부분 파일은 **하단에 `# 정답` 주석으로** 정답이 같이 들어 있음.
- 정답 안 보이는 파일은 강의 메인 폴더 (`../`)의 같은 번호 파일을 참고.
- 막히면 강사한테 메모해두기. 다음 회차 시작에 같이 봅니다.
