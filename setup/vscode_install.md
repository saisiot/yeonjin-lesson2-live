# VSCode 설치 가이드

> **이걸 왜 깔까요?** 1회차 비유로 "글 쓰는 종이의 진화"입니다. 메모장처럼 그냥 글만 쓰는 게 아니라, 코드 색깔을 알아서 칠해주고, 실행 버튼이 옆에 붙어 있고, 터미널이 아래에 같이 열리는 작업창이에요. 2회차에 파이썬 한 줄씩 칠 때, Git 명령 굴릴 때 — 다 이 창 하나에서 합니다.

---

## 1단계 — 다운로드

공식 사이트로만 받으세요. 비슷한 이름의 가짜 사이트가 있어서 검색 결과 첫 줄이 광고면 피하시는 게 좋습니다.

**공식 다운로드 페이지**: https://code.visualstudio.com/

페이지에 들어가면 본인 OS에 맞는 큰 파란 버튼이 자동으로 보입니다.

- **Mac 사용자**: "Download for macOS" 버튼이 보입니다. 클릭하면 `.zip` 파일이 다운로드됩니다.
- **Windows 사용자**: "Download for Windows" 버튼이 보입니다. 클릭하면 `.exe` 설치 파일이 다운로드됩니다.

[스크린샷: VSCode 공식 사이트 첫 화면, 가운데 큰 다운로드 버튼이 보이는 모습]

## 2단계 — 설치

### Mac

1. 다운로드 받은 `.zip` 파일을 더블클릭하면 자동으로 풀려서 **Visual Studio Code.app** 파일이 같은 폴더에 생깁니다.
2. 그 앱 파일을 **응용 프로그램(Applications) 폴더로 끌어다 놓습니다**. 이게 Mac에서 앱을 "설치"하는 방식이에요.
3. 응용 프로그램 폴더에서 VSCode를 더블클릭해서 실행합니다.
4. 처음 실행하면 "인터넷에서 받은 앱입니다. 정말 여시겠습니까?" 경고가 뜰 수 있어요. **"열기" 클릭하시면 됩니다**.

[스크린샷: Mac 응용 프로그램 폴더에 VSCode 앱이 들어가 있는 모습]

### Windows

1. 다운로드 받은 `.exe` 파일을 더블클릭합니다.
2. 설치 마법사가 뜹니다. 약관 동의 → 다음 → 다음 순서로 진행하시면 됩니다. **대부분 기본값 그대로** 두셔도 됩니다.
3. 중간에 "Select Additional Tasks" 화면이 나오면 다음 항목들을 체크하시는 걸 권합니다.
   - ✅ "Add to PATH" (보통 기본 체크되어 있음 — 그대로 두세요)
   - ✅ "Register Code as an editor for supported file types"
4. 설치 완료 후 자동 실행됩니다.

> 참고로 다운로드 페이지에 보면 "User Installer"와 "System Installer"가 있을 수 있어요. **일반 사용자는 User Installer**가 정답입니다. 관리자 권한이 필요 없어서 그게 더 편해요.

[스크린샷: Windows 설치 마법사의 Additional Tasks 화면]

## 3단계 — 첫 실행 + 한국어 언어 팩 설치

VSCode를 처음 켜면 영어로 보입니다. 한국어로 바꿔두는 게 편합니다.

1. VSCode 왼쪽 사이드바에 **사각형 4개 모양 아이콘**이 보입니다. 그게 **Extensions(확장)** 메뉴예요. 클릭하세요.
2. 위쪽 검색창에 `Korean Language Pack` 이라고 칩니다.
3. **"Korean Language Pack for Visual Studio Code"** (Microsoft 제공) 항목을 찾아서 **Install** 버튼을 누릅니다.
4. 설치가 끝나면 오른쪽 아래에 작은 알림이 뜹니다 — "Restart" 버튼을 눌러서 VSCode를 재시작하세요.
5. 재시작하면 메뉴들이 한국어로 보입니다.

> **막힐 때**: 재시작 안내가 안 뜨면 VSCode를 직접 닫았다가 다시 여시면 됩니다. 그래도 영어로 보이면 화면 왼쪽 아래 톱니바퀴 → "Command Palette" → `Configure Display Language` 검색 → `ko` 선택.

[스크린샷: Extensions 패널에서 Korean Language Pack을 찾아 Install 버튼이 보이는 모습]

## 4단계 — Python 확장 설치

1회차 비유로 VSCode가 "글 쓰는 종이"라면, 종이 위에 파이썬 글을 쓸 때 알아서 색깔 칠해주고 실행 버튼 붙여주는 도우미가 별도로 필요합니다. 그게 Python 확장이에요.

1. 다시 왼쪽 **Extensions(확장)** 메뉴로 들어갑니다.
2. 검색창에 `Python` 이라고 칩니다.
3. **"Python"** (Microsoft 제공, 다운로드 수가 가장 많은 항목)을 찾아서 **설치** 버튼을 누릅니다.
4. 설치가 끝나면 자동으로 함께 들어오는 **Pylance** 같은 보조 확장도 같이 깔립니다 — 그대로 두세요.

[스크린샷: Extensions 검색창에 'Python'을 친 결과 첫 번째 Microsoft 항목]

> 참고: 파이썬 자체는 별도로 설치합니다 (python_install.md 참고). 여기서 깐 건 "VSCode가 파이썬을 알아보게 해주는 도우미"일 뿐이에요.

## 5단계 — 터미널 열기

코드만 쓰는 게 아니라 명령어도 칠 수 있어야 합니다. VSCode 안에 터미널이 같이 들어 있어요.

- 상단 메뉴 → **터미널 → 새 터미널** (또는 영문판: **Terminal → New Terminal**)
- 단축키
  - **Mac**: `⌃` (Control) + `` ` `` (백틱, 숫자 1 왼쪽 키)
  - **Windows**: `Ctrl` + `` ` ``

VSCode 화면 아래쪽에 검은 영역이 열리고, 거기에 글씨를 칠 수 있게 되면 성공입니다.

[스크린샷: VSCode 화면 아래쪽에 터미널 패널이 열린 모습]

## 6단계 — 확인

지금까지 잘 따라오셨다면 다음 4가지가 모두 OK여야 합니다.

- [ ] VSCode가 켜진다
- [ ] 메뉴가 한국어로 보인다
- [ ] 왼쪽 Extensions 메뉴에서 "Python"이 설치됨으로 표시된다
- [ ] 화면 아래쪽에 터미널이 열린다

4개 다 OK면 VSCode 설치는 끝입니다. 다음은 Python 설치예요.

## 막힐 때 — 자주 보는 자리

- **한국어 팩 설치 후에도 영어로 보임** → VSCode 완전히 닫았다가 다시 여세요. 그래도 안 되면 톱니바퀴 → Command Palette → `Configure Display Language` → `ko` 선택.
- **다운로드 페이지에서 어떤 버튼을 눌러야 할지 헷갈림** → 페이지가 자동으로 OS를 감지해서 큰 파란 버튼 하나만 보여줍니다. 그것만 누르세요.
- **Mac에서 "확인되지 않은 개발자가 만든 앱이라 열 수 없습니다" 경고** → 시스템 설정 → 개인정보 보호 및 보안 → 화면 아래쪽 "확인 없이 열기" 한 번 클릭. 그 다음 다시 VSCode 더블클릭.
- **Windows에서 설치 후 시작 메뉴에 안 보임** → 다시 켜기. 또는 검색창에 `Visual Studio Code` 검색.
